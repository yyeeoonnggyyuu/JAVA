public static void main(String[] args) { // 코드 작성}
public: 메서드가 어디서든 접근 가능하다는 것을 의미합니다.
static: 객체를 생성하지 않고도 메서드를 호출할 수 있음을 의미합니다.
void: 메서드가 아무 값도 반환하지 않음을 의미합니다.
main: Java가 프로그램 실행 시 호출하는 메서드 이름입니다.
String[] args: 프로그램 실행 시 전달받을 수 있는 문자열 인자들을 나타냅니다.
